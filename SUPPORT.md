# Support

For public repositories, use GitHub issues when the request is about an open-source project and can be discussed publicly.

For private product, partnership, security, or account-related requests, contact:

- info@cdli.ai
- https://cdli.ai/work-with-us
- https://cdli.ai/tr/birlikte-calisalim

Please include the repository or product name, the environment affected, and the outcome you expected.

Public support and sponsorship conversations can start from the work-with-us page. Security reports should always stay private and follow `SECURITY.md`.
